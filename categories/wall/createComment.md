# Метод Wall.CreateComment
Проверяет ссылку для указания источника.

Страница документации ВКонтакте [wall.createComment](https://vk.com/dev/wall.createComment).

Этот метод можно вызвать с ключом доступа пользователя, полученным в [Standalone-приложении](https://vk.com/dev/standalone) через [Implicit Flow](https://vk.com/dev/implicit_flow_user).  
Этот метод можно вызвать с [ключом доступа сообщества](https://vk.com/dev/access_token).   
Требуются [права доступа](https://vk.com/dev/permissions) wall.  

## Синтаксис
``` csharp
public long CreateComment(WallCreateCommentParams @params)
```

## Параметры
+ **OwnerId** - Идентификатор пользователя или сообщества, на чьей стене находится запись, к которой необходимо добавить комментарий. По умолчанию идентификатор текущего пользователя.
+ **PostId** - Идентификатор записи на стене пользователя или сообщества. Обязательный параметр.
+ **FromGroup** - Идентификатор сообщества, от имени которого публикуется комментарий. По умолчению 0.
+ **Message** - Текст комментария. Обязательный параметр, если не передан параметр **attachments**.
+ **ReplyToComment** - Идентификатор комментария, в ответ на который должен быть добавлен новый комментарий.
+ **Attachments** - Список медиавложений, приложенных к комментарию
+ **StickerId** - Идентификатор стикера.
+ **Guid** - Уникальный идентификатор, предназначенный для предотвращения повторной отправки одинакового комментария.
+ **CaptchaSid** - Идентификатор капчи
+ **CaptchaKey** - Текст капчи, который ввел пользователь

## Результат
В случае успеха метод вернет **true**.
